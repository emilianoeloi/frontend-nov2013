HTML

Uma página básica em HTML é um arquivo de texto simples contendo o texto da página e os comandos HTML que definem a formatação das mesmas.

CSS

A vantagem de se usar o código CSS em um arquivo.css é que caso voce precise ou queira alterar alguma coisa. As alterações serão feitas em todas as páginas que estiverem usando o arquivo.css.
